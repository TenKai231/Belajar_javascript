# Belajar JavaScript

Repositori ini berisi kumpulan materi dan latihan belajar JavaScript, mulai dari dasar hingga konsep lanjutan.

## Struktur Modul

| Modul | Topik |
|-------|-------|
| [Module_1](./Module_1) | Variabel, tipe data, konversi tipe, dan operator |
| [Module_2](./Module_2) | Fungsi, parameter default, dan arrow function |
| [Module_3](./Module_3) | Object, destructuring, dan manipulasi properti |
| [Module_4](./Module_4) | Spread operator, array, dan string |
| [Module_5](./Module_5) | Kondisional: `if/else`, `switch`, dan ternary operator |
| [Module_6](./Module_6) | ES Module: `import` dan `export` |
| [Module_7](./Module_7) | Class dan OOP (Object-Oriented Programming) |
| [Module_8](./Module_8) | Array, fungsi higher-order (`map`), dan pure function |
| [Module_9](./Module_9) | Asynchronous JavaScript: `setTimeout` dan Promise |
| [Module_10](./Module_10) | Dynamic typing dan TypeScript dasar |
| [Module_11](./Module_11) | TypeScript, ESLint, Flow, dan pengujian dengan Jest |

## Final Assessment

- `optional-05-recursive` — Rekursi (opsional)
- `optional-06-full-coverage-testing` — Pengujian dengan cakupan penuh (opsional)

## Cara Menjalankan

Pastikan [Node.js](https://nodejs.org/) sudah terinstal.

```bash
# Jalankan file JavaScript biasa
node Module_1/main.js

# Jalankan file ES Module (.mjs)
node Module_6/main.mjs

# Jalankan pengujian (di folder yang memiliki package.json)
cd Module_11
npm install
npm test
```
